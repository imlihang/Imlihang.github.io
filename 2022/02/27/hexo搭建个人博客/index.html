hexo搭建个人博客

发表于 2022-02-27 更新于 2023-03-12 分类于 Hexo

下载 Node.js

https://nodejs.org/en/

安装hexo

npm install -g hexo-cli

hexo命令

hexo init

hexo g 生成静态页

hexo s 本地启动 预览使用

hexo n “helloworld” 创建一篇文章

发布Github

  • 新建 repository repository name 必须是 {use_name}.github.io

  • 安装插件 npm install –save hexo-deployer-git

  • 修改 _config.yml

​ deploy:
​ type: git
​ repo: [https://github.com/ {use_name}/ {use_name}.github.io.git](https://github.com/ {use_name}/ {use_name}.github.io.git)
​ brach: master

  • hexo d 部署

一些报错问题

ERROR Deployer not found:

npm install –save hexo-deployer-git

*** Please tell me who you are.

在 .git目录下打开配置文件config，添加

[user]

email = xxx.com

name = xxxx
